## <a name="functions"></a>函数

以下标量函数对数组输入值执行操作，并返回数值、布尔值或数组值：

* [ARRAY_CONTAINS](sql-query-array-contains.md)
* [ARRAY_LENGTH](sql-query-array-length.md)
