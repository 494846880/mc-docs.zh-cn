## <a name="release-and-retirement-dates"></a>发布日期和停用日期
Azure 会在停用 SDK 时至少提前 12 个月  发出通知，以便用户顺利转换为更高版本/受支持版本。

新特性和功能以及优化仅添加到当前 SDK，因此建议始终尽早升级到最新的 SDK 版本。
