# <a name="azure-policy-regulatory-compliance-controls-for-azure-database-for-mariadb"></a>Azure Database for MariaDB 的 Azure Policy 法规遵从性控制措施

Azure Policy 中的法规符合性为与不同符合性标准相关的“符合域”和“安全控制措施”提供 Azure 创建和管理的计划定义，称为“内置” 。 此页列出 Azure Database for MariaDB 的“符合域”和“安全控制措施” 。 可以分别为“安全控件”分配内置项，以帮助 Azure 资源符合特定的标准。
