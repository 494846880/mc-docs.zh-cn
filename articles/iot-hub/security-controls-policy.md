# <a name="azure-policy-regulatory-compliance-controls-for-azure-iot-hub"></a>Azure IoT 中心的 Azure Policy 合规性控制

Azure Policy 中的法规符合性为与不同符合性标准相关的“符合域”和“安全控件”提供 Microsoft 创建和管理的计划定义，称为“内置” 。 此页列出 Azure IoT 中心的“符合域”和“安全控件”。 可以分别为“安全控件”分配内置项，以帮助 Azure 资源符合特定的标准。
