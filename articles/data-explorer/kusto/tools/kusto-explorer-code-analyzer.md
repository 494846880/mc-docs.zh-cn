# <a name="kusto-explorer-code-analyzer"></a>Kusto Explorer 代码分析器

Kusto.Explorer 提供代码分析器实用程序，可分析当前查询并输出一组适用的改进建议。

使用 `Ctrl`+`F6` 对活动查询运行分析器。
