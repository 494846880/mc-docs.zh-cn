# <a name="sync-kusto"></a>同步 Kusto

同步 Kusto 是一种工具，使用户能够同步各种 Kusto 架构实体，如表架构和存储的函数。 这种同步在本地文件系统、Kusto 数据库和 Azure Dev Ops 存储库之间进行。

[GitHub](https://github.com/microsoft/synckusto) 上提供了 Sync Kusto。
