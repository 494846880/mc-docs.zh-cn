# <a name="exp10"></a>exp10()

x 的以 10 为底的指数函数，表示 10 的 x 次幂：10^x。  

## <a name="syntax"></a>语法

`exp10(`*x*`)`

## <a name="arguments"></a>参数

* x：实数，指数的值。

## <a name="returns"></a>返回

* x 的指数值。
* 有关自然对数（以 10 为底），请参阅 [log10()](log10-function.md)。
* 有关以 e 为底和以 2 为底的对数的指数函数，请参阅 [exp()](exp-function.md)、[exp2()](exp2-function.md)
