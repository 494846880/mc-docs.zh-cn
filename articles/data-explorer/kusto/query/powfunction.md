# <a name="pow"></a>pow()

返回幂运算的结果

## <a name="syntax"></a>语法

`pow(`*base*`,` *exponent* `)`

## <a name="arguments"></a>参数

* base：底数值。
* exponent：指数值。

## <a name="returns"></a>返回

返回底数的幂指数：底数^指数。
