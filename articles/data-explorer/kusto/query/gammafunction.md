# <a name="gamma"></a>gamma()

计算 gamma 函数

## <a name="syntax"></a>语法

`gamma(`*x*`)`

## <a name="arguments"></a>参数

* x：gamma 函数的参数

## <a name="returns"></a>返回

* x 的 gamma 函数。
* 若要计算对数 gamma 函数，请参阅 [loggamma()](loggammafunction.md)。
