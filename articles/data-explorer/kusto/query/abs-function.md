# <a name="abs"></a>abs()

计算输入的绝对值。  

## <a name="syntax"></a>语法

`abs(`x`)`

## <a name="arguments"></a>参数

* x：整数或实数，或者时间跨度值。

## <a name="returns"></a>返回

* x 的绝对值。
