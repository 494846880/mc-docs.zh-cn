# <a name="tan"></a>tan()

返回正切函数。

## <a name="syntax"></a>语法

`tan(`*x*`)`

## <a name="arguments"></a>参数

* x：一个实数。

## <a name="returns"></a>返回

* `tan(x)` 的结果
