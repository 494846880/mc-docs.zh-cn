# <a name="sin"></a>sin()

返回正弦函数。

## <a name="syntax"></a>语法

`sin(`*x*`)`

## <a name="arguments"></a>参数

* x：一个实数。

## <a name="returns"></a>返回

* `sin(x)` 的结果
