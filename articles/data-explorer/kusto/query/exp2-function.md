# <a name="exp2"></a>exp2()

x 的以 2 为底数的指数函数，表示 2 的 x 次幂：2^x。  

## <a name="syntax"></a>语法

`exp2(`*x*`)`

## <a name="arguments"></a>参数

* x：实数，指数的值。

## <a name="returns"></a>返回

* x 的指数值。
* 有关自然对数（以 2 为底），请参阅 [log2()](log2-function.md)。
* 关于以 e 为底和以 10 为底的对数的指数函数，请参阅 [exp()](exp-function.md)、[exp10()](exp10-function.md)
