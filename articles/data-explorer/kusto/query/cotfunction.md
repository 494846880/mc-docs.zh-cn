# <a name="cot"></a>cot()

计算指定角度的三角函数余切（以弧度为单位）。

## <a name="syntax"></a>语法

`cot(`*x*`)`

## <a name="arguments"></a>参数

* x：实数。

## <a name="returns"></a>返回

* `x` 的余切函数值
