# <a name="atan"></a>atan()

返回其正切为指定数字的角度（[`tan()`](tanfunction.md) 的反运算）。

## <a name="syntax"></a>语法

`atan(`*x*`)`

## <a name="arguments"></a>参数

* x：一个实数。

## <a name="returns"></a>返回

* `x` 的反正切的值
