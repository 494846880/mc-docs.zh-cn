# <a name="loggamma"></a>loggamma()

计算伽马函数绝对值的对数

**语法**

`loggamma(`*x*`)`

**参数**

* x：伽马函数的参数

**返回**

* 返回 x 的伽马函数绝对值的自然对数。
* 若要计算伽马函数，请参阅 [gamma()](gammafunction.md)。
