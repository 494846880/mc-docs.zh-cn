# <a name="exp"></a>exp()

x 的以 e 为底的指数函数，表示 e 的 x 次幂：e^x。  

## <a name="syntax"></a>语法

`exp(`*x*`)`

## <a name="arguments"></a>参数

* x：实数，指数的值。

## <a name="returns"></a>返回

* x 的指数值。
* 有关自然对数（以 e 为底），请参阅 [log()](log-function.md)。
* 有关以 2 为底和以 10 为底的对数的指数函数，请参阅 [exp2()](exp2-function.md)、[exp10()](exp10-function.md)
