# <a name="azure-data-explorer-java-sdk"></a>Azure 数据资源管理器 Java SDK

Kusto Java 客户端库提供了使用 Java 查询 Azure 数据资源管理器群集的功能。
Azure 数据资源管理器 Java SDK 在 [azure-kusto-java](https://github.com/Azure/azure-kusto-java) 中提供。
