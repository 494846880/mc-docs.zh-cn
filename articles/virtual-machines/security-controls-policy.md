# Azure Policy Regulatory Compliance controls for Azure Virtual Machines 

Azure Policy 中的法规符合性为与不同符合性标准相关的“符合域”和“安全控制措施”提供 Azure 创建和管理的计划定义，称为“内置” 。 此页列出 Azure 虚拟机的“符合域”和“安全控制措施” 。 可以分别为“安全控制”分配内置项，以帮助 Azure 资源符合特定的标准。
