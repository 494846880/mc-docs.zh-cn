# <a name="azure-cli-samples-for-azure-database-for-mysql"></a>用于 MySQL 的 Azure 数据库的 Azure CLI 示例

|**缩放服务器**||
| [缩放服务器](./scripts/sample-scale-server.md) | 此 Azure CLI 脚本可扩大或缩小用于 MySQL 服务器的单个 Azure 数据库，以允许更改性能要求。 |
|**更改服务器配置**||
| [更改服务器配置](./scripts/sample-change-server-configuration.md) | 更改适用于 MySQL 服务器的单个 Azure 数据库配置的 Azure CLI 脚本。 |
|**还原服务器**||
| [还原服务器](./scripts/sample-point-in-time-restore.md) | 用于将单个 Azure Database for MySQL 服务器还原到前一个时间点的 Azure CLI 脚本。 |
|**操作服务器日志**||
| [启用和下载服务器日志](./scripts/sample-server-logs.md) | 用于启用和下载单个 Azure Database for MySQL 服务器的服务器日志的 Azure CLI 脚本。 |
