## <a name="run-the-speech-cli"></a>运行语音 CLI

现在，可以运行语音 CLI，以从文本合成语音。

在命令行中，更改为包含语音 CLI 二进制文件的目录，然后键入：

```bash
spx synthesize --text "The speech synthesizer greets you!"
```

语音 CLI 将通过计算机扬声器生成英语的自然语言。
