此实体类别包括标识的财务信息和正式形式。 从模型版本 `2019-10-01` 开始提供。 下面列出了子类型。 

### <a name="financial-account-identification"></a>财务帐户标识

| 子类型名称               | 说明                                                                |
|----------------------------|----------------------------------------------------------------------------|
| ABA 路由号码        | 美国银行家协会 (ABA) 转账路由号码。                  |
| SWIFT 代码                 | 付款说明信息中的 SWIFT 代码。                           |
| 信用卡                | 信用卡卡号。                                                       |
| 国际银行帐号 (IBAN)                  | 付款说明信息中的 IBAN 代码。                            |
