## <a name="clean-up-resources"></a>清理资源

不再需要 LUIS 应用时，请将其删除。 为此，请在左上角的菜单中选择“我的应用”。 在应用列表中选择应用名称右侧的省略号 (...)，然后选择“删除”。 在弹出的“删除应用?”对话框中，选择“确定” 。
