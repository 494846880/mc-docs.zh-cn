# <a name="set-up-single-sign-on"></a>设置单一登录

我们在 Azure Databricks 中为所有客户提供单一登录 (SSO)，采用 Azure Active Directory 支持的登录形式。 请参阅 [Azure Active Directory 无缝单一登录](/active-directory/connect/active-directory-aadconnect-sso)。
