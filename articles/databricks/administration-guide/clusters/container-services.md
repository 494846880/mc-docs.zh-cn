# <a name="enable-container-services"></a>启用容器服务

若要在群集上使用[自定义容器](../../clusters/custom-containers.md)，工作区管理员必须启用 Databricks 容器服务，如下所示：

1. 转到[管理控制台](../admin-console.md)。
2. 单击“高级”  选项卡。
3. 单击“容器服务”旁边的“启用”按钮 。
4. 刷新页面。
