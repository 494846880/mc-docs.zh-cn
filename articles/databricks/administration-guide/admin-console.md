# <a name="access-the-admin-console"></a>访问管理控制台

若要访问管理控制台，请转到工作区 UI 的右上方，单击 ![“帐户”图标](../_static/images/account-settings/account-icon.png) 用户帐户图标，然后选择“管理控制台”。

你必须是具有[管理员权限](users-groups/users.md)的用户才能访问管理控制台。
