# <a name="databricks-rest-api"></a>Databricks REST API

Azure Databricks 有两个用于执行不同任务的 REST API：2.0 和 1.2。 对于常规管理，请使用 REST API 2.0。 REST API 1.2 用于直接在 Azure Databricks 上运行命令。
