# <a name="databricks-runtime-54-unsupported"></a>Databricks Runtime 5.4（不受支持）

Databricks 于 2019 年 6 月发布了此映像。

以下发行说明提供了由 Apache Spark 支持的 Databricks Runtime 5.4 的相关信息。

## <a name="new-features"></a>新功能

### <a name="delta-lake-on-azure-databricks-adds-auto-optimize-public-preview"></a>Azure Databricks 上的 Delta Lake 增加了自动优化（公共预览版）

现在，将数据写入云存储时，你需要压缩文件以获得最佳 I/O 性能。 你必须考虑正确的文件大小、压缩文件的频率、要使用的群集的大小，等等。 为了解决此类问题，我们非常高兴地介绍通过 Azure Databricks 上的 Delta Lake 进行自动优化的功能。 每次写入到 Delta 表期间，我们都会自动确定正确的文件大小并压缩文件，因此你不必为优化存储布局担忧。 在写入期间，如果 `auto-optimize` 选项为 `true`，则 Azure Databricks 会自动确定是否需要优化以及是否优化小文件。 有关详细信息，请参阅[自动优化](../../delta/optimizations/auto-optimize.md)。

### <a name="library-utilities-generally-available"></a>库实用工具已正式发布

库实用工具现在已正式发布。 此功能可让你从笔记本会话范围内操作 Python 库。  请参阅[库实用工具](../../dev-tools/databricks-utils.md#dbutils-library)。

### <a name="databricks-connect-generally-available"></a>Databricks Connect 已正式发布

Databricks Connect 现在已正式发布。  此功能是一个 Spark 客户端库，可用来将你喜欢的 IDE（IntelliJ、Eclipse、PyCharm、RStudio）、笔记本服务器（Zeppelin、Jupyter）和其他自定义应用程序连接到 Azure Databricks 群集，并运行 Spark 代码。 请参阅 [Databricks Connect](../../dev-tools/databricks-connect.md)。

### <a name="binary-file-data-source"></a>二进制文件数据源

二进制文件数据源读取二进制文件并将每个文件转换为包含该文件的原始内容和元数据的单个记录。  你现在可以将任意文件有效地读入 Spark 数据帧，无需访问文件的内容。 如果要使用 Spark 加载大型文件（例如图像）来进行评分，这将特别有用。 请参阅[二进制文件](../../data/data-sources/binary-file.md)。

## <a name="improvements"></a>改进

**增量缓存**

现在，Lsv2 实例预配置了 [Delta 缓存](../../delta/optimizations/delta-cache.md)。 但是，默认情况下不会启用缓存。 有关如何启用 Delta 缓存的信息，请参阅[启用 Delta 缓存](../../delta/optimizations/delta-cache.md#enable-disable)。

**Azure Data Lake Storage Gen2 Azure AD 凭据直通身份验证**

现在支持 DBFS 通过 Azure AD 凭据直通身份验证装载 Azure Data Lake Storage (ADLS) Gen2 文件系统。 请参阅[使用凭据直通身份验证将 Azure Data Lake Storage 装载到 DBFS](../../security/credential-passthrough/adls-passthrough.md#aad-passthrough-dbfs)。

**已将 R 升级到 3.6.0**

R 基础映像已从 3.5.2 升级到 3.6.0。

## <a name="system-environment"></a>系统环境

* **操作系统**：Ubuntu 16.04.6 LTS
* **Java**：1.8.0_191
* **Scala**：2.11.12
* **Python**：2.7.12 适用于 Python 2 群集，3.5.2 适用于 Python 3 群集。 有关详细信息，请参阅 [Python 版本](../../clusters/configure.md#python-3)。
* **R**：R 版本 3.6.0 (2019-04-26)
* **GPU 群集**：安装了以下 NVIDIA GPU 库：
  * Tesla 驱动程序 375.66
  * CUDA 9.0
  * cuDNN 7.0

> [!NOTE]
>
> 尽管 Apache Spark 2.4 支持 Scala 2.12，但 Databricks Runtime 5.4 不支持它。
