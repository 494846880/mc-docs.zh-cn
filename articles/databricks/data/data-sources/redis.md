# <a name="redis"></a>Redis

[Redis](https://redis.io/) 是一种常用的键-值存储，快速且易用。 以下笔记本演示如何结合使用 Redis 与 Apache Spark。

> [!NOTE]
>
> 无法从运行 Databricks Runtime 7.0 或更高版本的群集访问此数据源，因为支持 Apache Spark 3.0 的 Redis 时序连接器不可用。
