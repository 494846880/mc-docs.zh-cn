# <a name="elasticsearch"></a>ElasticSearch

[ElasticSearch](https://www.elastic.co/elasticsearch) 是一种分布式 RESTful 搜索和分析引擎。

> [!NOTE]
>
> 无法从运行 Databricks Runtime 7.0 或更高版本的群集访问此数据源，因为支持 Apache Spark 3.0 的 ElasticSearch 连接器不可用。
