### <a name="prerequisite-check"></a>先决条件检查

+ 在终端或命令窗口中，运行 `func --version` 检查 Azure Functions Core Tools 的版本是否为 2.7.1846 或以上。

+ 运行 `az --version` 检查 Azure CLI 版本是否为 2.0.76 或以上。

+ 运行 `az login` 登录到 Azure 并验证活动订阅。
